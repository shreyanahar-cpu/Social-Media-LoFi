# My Figma Project(LoFi)

## Project Overview
This is my UI/UX design project created using Figma.

## Project Description

This project is a low-fidelity (Lo-Fi) UI/UX design of a social media mobile application created using Figma. The design focuses on providing a simple, clean, and user-friendly interface for users to connect, share posts, and interact with content.

The app includes basic features such as user login and signup, home feed, profile section, post creation, and navigation between different sections. The layout is designed to ensure easy navigation and a smooth user experience.

This project demonstrates the initial design stage where structure, layout, and user flow are prioritized over visual details.
